## N.I.B. icons  
  
Requires:  

```shell
sudo bash -c "$(curl -LSs https://github.com/dfmgr/installer/raw/master/install.sh)"
```

Automatic:

```shell
bash -c "$(curl -LSs https://github.com/iconmgr/N.I.B./raw/master/install.sh)"
```

Install:

```shell
sudo iconmgr install N.I.B.
```

Update:

```shell
sudo iconmgr update N.I.B.
```
